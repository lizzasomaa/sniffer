Программа на языке C для анализа сетевого трафика.  
Сниффер читает пакеты из PCAP-файлов или захватывает их в реальном времени с сетевого интерфейса, анализирует заголовки протоколов и распределяет пакеты по категориям.

## Возможности

- чтение пакетов из PCAP-файла или директории с PCAP-файлами
- захват трафика в реальном времени с сетевого интерфейса
- анализ TCP и UDP пакетов
- выделение FTP Control и FTP Data соединений
- сборка завершённых TCP-сессий
- фильтрация некоторых UDP-пакетов
- многопоточная обработка пакетов
- распределение пакетов по отдельным PCAP-файлам

## Архитектура

Программа состоит из нескольких модулей:

- **process.c** — основной анализ сетевых пакетов
- **live_process.c** — захват трафика с сетевого интерфейса
- **queue.c** — потокобезопасная очередь пакетов
- **tcp_sessions.c** — сборка завершённых TCP-сессий
- **ftp.c** — обработка FTP Control/Data соединений

Для записи пакетов используются отдельные потоки-обработчики.

## Требования

- Linux
- gcc
- libpcap
- pthread
- cmake или make
